# AgentTrail

Local web viewer for Claude Code session history across multiple directories.

## Quick Reference

- **Runtime:** Bun
- **Install:** `bun install`
- **Dev:** `bun run dev`
- **Start:** `bun run start`
- **Build:** `bun run build`
- **Test:** `bun run test`
- **Unit:** `bun run test:unit`
- **API:** `bun run test:api`
- **E2E:** `bun run test:e2e`
- **E2E UI:** `bun run test:e2e:ui`
- **Coverage:** `bun test --coverage --coverage-reporter=lcov tests/unit tests/api`
- **Typecheck:** `bun run typecheck`
- **Lint:** `bun run check`
- **Format:** `bun run format`
